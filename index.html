Editor Controls

E - Toggle Editor

1 - Block

2 - Spike

3 - Start Block

4 - Goal Block

Left Click - Select / Place

Right Click - Delete

Ctrl + Click - Multi-Select

WASD - Move Camera

Arrow Keys - Rotate

Ctrl + C - Copy

Ctrl + Z - Undo

Ctrl + Y - Redo

Ctrl + S - Save Level

Ctrl + L - Load Level

Ctrl + Shift + R - Reset Level

Backspace - Delete Selected

Esc - Deselect


O - Toggle Controls Page
